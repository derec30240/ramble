# Triage 标签

各 skill 用五个规范 triage 角色交流。本文件把这些角色映射到本仓库 issue tracker 实际使用的标签字符串。

| skill 中的角色名  | 本仓库标签        | 含义                         |
| ----------------- | ----------------- | ---------------------------- |
| `needs-triage`    | `needs-triage`    | 待维护者评估                 |
| `needs-info`      | `needs-info`      | 等待报告者补充信息           |
| `ready-for-agent` | `ready-for-agent` | 规格完备，可交给智能体执行   |
| `ready-for-human` | `ready-for-human` | 需要人类实现                 |
| `wontfix`         | `wontfix`         | 不予处理                     |

当某个 skill 提到某个角色（例如"打上 AFK-ready 标签"）时，使用上表右列对应的标签字符串。

若日后更换词表，直接修改右列即可。
